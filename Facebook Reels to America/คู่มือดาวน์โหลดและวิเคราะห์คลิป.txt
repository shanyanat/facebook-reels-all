ส่วนที่ 1: การติดตั้งแบบง่ายโดยใช้ Codex

ใช้วิธีนี้เมื่อตั้งค่าเครื่องใหม่ หรือเครื่องสำรองที่ยังไม่เคยใช้ระบบนี้

1. เปิด Codex ในโฟลเดอร์โปรเจกต์นี้:

C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels All\Facebook Reels to America

2. พิมพ์บอก Codex ว่า:

setup clip downloader

3. Codex จะตรวจให้ว่าเครื่องมีเครื่องมือครบหรือยัง:

- yt-dlp ใช้สำหรับดาวน์โหลดคลิป
- ffmpeg เป็นเครื่องมือเสริม ใช้เฉพาะตอนต้องดึงภาพตัวอย่างจากวิดีโอเพื่อช่วยวิเคราะห์

4. ถ้ายังไม่มี yt-dlp ให้ติดตั้งด้วยคำสั่งนี้:

py -m pip install -U yt-dlp

5. ถ้ายังไม่มี ffmpeg และต้องการให้ Codex วิเคราะห์จากภาพตัวอย่างวิดีโอ ให้ติดตั้งด้วยคำสั่งนี้:

winget install Gyan.FFmpeg

6. หลังติดตั้งเสร็จ ให้ตรวจด้วยคำสั่ง:

py -m yt_dlp --version
ffmpeg -version

ถ้าทั้งสองคำสั่งแสดงข้อมูลเวอร์ชัน แปลว่าเครื่องพร้อมใช้งานแล้ว


ส่วนที่ 2: เตรียมไฟล์ url_clip.txt

เปิดไฟล์นี้:

C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels All\Facebook Reels to America download clip\url_clip.txt

รูปแบบไฟล์ควรเป็นแบบนี้:

Channel Name: 1-page-Strange-Frontiers

1.
URL: https://www.instagram.com/reel/xxxxx/
Status:
Topic:

2.
URL: https://www.facebook.com/reel/xxxxx
Status:
Topic:

3.
URL:
Status:
Topic:

วิธีใส่ลิงก์:

- ใส่ 1 URL ต่อ 1 หมายเลข
- ใช้ได้ทั้ง Instagram Reels และ Facebook Reels/Video
- อย่าลบ Channel Name
- Status และ Topic ปล่อยว่างไว้ก่อน
- ถ้ามีมากกว่า 3 คลิป ให้เพิ่ม 4., 5., 6. ต่อไปได้


ส่วนที่ 3: คำสั่งใช้งานหลักสำหรับดาวน์โหลด

เปิด Warp หรือ PowerShell แล้วรัน 2 บรรทัดนี้:

cd "C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels All\Facebook Reels to America"
py process_clip_batch.py

คำสั่งนี้จะทำงานให้โดยอัตโนมัติ:

- อ่านไฟล์ url_clip.txt
- อ่านชื่อ Channel Name
- สร้างหรือใช้โฟลเดอร์ช่องเดิม
- ดาวน์โหลดคลิปตามลำดับในไฟล์
- ใส่ Status: Downloaded ให้คลิปที่โหลดสำเร็จ
- ล้างไฟล์ช่วยวิเคราะห์เก่าออก เพื่อให้โฟลเดอร์สุดท้ายเหลือเฉพาะคลิปวิดีโอ

หลังดาวน์โหลดเสร็จ คลิปจะอยู่ในโฟลเดอร์นี้:

C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels All\Facebook Reels to America download clip\(Channel Name)


ส่วนที่ 4: วิเคราะห์คลิปด้วย Codex

หลังรันคำสั่งดาวน์โหลดเสร็จ ให้กลับมาที่ Codex แล้วพิมพ์:

analyze downloaded clip batch

Codex จะช่วยทำงานส่วนที่ต้องใช้ AI:

- ดูคลิปหรือสร้างภาพตัวอย่างชั่วคราวเมื่อจำเป็น
- เขียน Topic ภาษาไทย 1-2 บรรทัด
- ตั้งชื่อไฟล์ภาษาไทยแบบสั้นให้เข้าใจง่าย
- เปลี่ยนชื่อไฟล์เป็นรูปแบบ:

1-ชื่อสั้นของคลิป.mp4
2-ชื่อสั้นของคลิป.mp4
3-ชื่อสั้นของคลิป.mp4

ตัวอย่าง:

1-แปลงใต้บันไดเป็นมุมทำงาน.mp4
2-ทำแผงลายนูนพ่นสีเมทัลลิก.mp4
3-ทำแผ่นทางเดินจากฝาขวด.mp4

หมายเหตุ:

โดยปกติ โฟลเดอร์ช่องควรเหลือเฉพาะไฟล์ .mp4 เท่านั้น
ไม่จำเป็นต้องเก็บโฟลเดอร์ _frames หรือไฟล์ .info.json
ถ้า Codex ต้องใช้ไฟล์ช่วยวิเคราะห์ จะสร้างแบบชั่วคราวและลบทิ้งหลังใช้งาน


ส่วนที่ 5: ล้างข้อมูลเก่าเมื่อใช้งานเสร็จ

เมื่องานชุดนั้นเสร็จแล้ว และต้องการเตรียมไฟล์สำหรับชุดใหม่ ให้กลับมาที่ Codex แล้วพิมพ์:

clear clip batch

Codex จะล้างข้อมูลเก่าใน url_clip.txt:

- ลบ URL เก่า
- ลบ Status เก่า
- ลบ Topic เก่า
- เก็บ Channel Name ไว้เหมือนเดิม

ถ้าต้องการลบคลิปวิดีโอเก่าในโฟลเดอร์ช่องด้วย ให้พิมพ์ให้ชัดเจนว่า:

clear clip batch and delete old clips

หมายเหตุ: การลบคลิปเป็นการลบไฟล์จริง ควรใช้เมื่อแน่ใจว่าใช้คลิปชุดนั้นเสร็จแล้ว


ส่วนที่ 6: วิธีใช้แบบสั้นที่สุด

ทุกครั้งที่ต้องการดาวน์โหลดชุดใหม่:

1. เปิด url_clip.txt
2. ใส่ URL ใหม่
3. เปิด Warp หรือ PowerShell
4. รัน:

cd "C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels All\Facebook Reels to America"
py process_clip_batch.py

5. กลับมาที่ Codex แล้วพิมพ์:

analyze downloaded clip batch

6. เมื่องานเสร็จและต้องการล้างข้อมูลเก่า ให้พิมพ์:

clear clip batch

หรือถ้าต้องการลบคลิปเก่าด้วย:

clear clip batch and delete old clips


ส่วนที่ 7: หมายเหตุสำคัญ

- ดาวน์โหลดเฉพาะคลิปสาธารณะ หรือคลิปที่คุณมีสิทธิ์ใช้เท่านั้น
- ไม่ควรใช้กับคลิปส่วนตัว กลุ่มปิด หรือคลิปที่ต้องข้ามระบบล็อกอิน/สิทธิ์เข้าถึง
- ถ้าดาวน์โหลดล้มเหลว อาจเป็นเพราะลิงก์ต้องล็อกอิน หรือแพลตฟอร์มจำกัดการเข้าถึง
- เครื่องที่ไม่มี Codex ยังสามารถรัน py process_clip_batch.py เพื่อดาวน์โหลดได้
- แต่การเขียน Topic ภาษาไทยและการตั้งชื่อไฟล์แบบเข้าใจเนื้อหา ยังควรใช้ Codex
