ส่วนที่ 1: การติดตั้งแบบง่ายโดยใช้ Codex

ใช้วิธีนี้เมื่อตั้งค่าเครื่องใหม่ หรือเครื่องสำรองที่ยังไม่เคยใช้ระบบนี้

1. เปิด Codex ในโฟลเดอร์โปรเจกต์นี้:

C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels All\Facebook Reels to America

2. พิมพ์บอก Codex ว่า:

setup clip downloader

3. Codex จะตรวจให้ว่าเครื่องมีเครื่องมือครบหรือยัง:

- yt-dlp ใช้สำหรับดาวน์โหลดคลิป
- ffmpeg เป็นเครื่องมือเสริม ใช้เฉพาะตอนต้องดึงภาพตัวอย่างจากวิดีโอเพื่อช่วยวิเคราะห์

4. ถ้ายังไม่มี yt-dlp ให้ติดตั้งด้วยคำสั่งนี้:

py -m pip install -U yt-dlp

5. ถ้ายังไม่มี ffmpeg และต้องการให้ Codex วิเคราะห์จากภาพตัวอย่างวิดีโอ ให้ติดตั้งด้วยคำสั่งนี้:

winget install Gyan.FFmpeg

6. หลังติดตั้งเสร็จ ให้ตรวจด้วยคำสั่ง:

py -m yt_dlp --version
ffmpeg -version

ถ้าทั้งสองคำสั่งแสดงข้อมูลเวอร์ชัน แปลว่าเครื่องพร้อมใช้งานแล้ว


ส่วนที่ 2: ใช้งานผ่านหน้าต่าง Clip Batch Control

ตอนนี้ไม่จำเป็นต้องเปิด url_clip.txt เพื่อวางลิงก์เองแล้ว
ให้ใช้หน้าต่างโปรแกรมแทน เพื่อป้องกันการเขียนทับข้อมูลช่องอื่น

เปิด Warp หรือ PowerShell แล้วรัน:

cd "C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels All\Facebook Reels to America"

หน้าต่างนี้จะจัดการไฟล์กลางนี้ให้อัตโนมัติ:

C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels All\Facebook Reels to America download clip\url_clip.txt

หลักการใหม่:

- url_clip.txt เป็นไฟล์กลางไฟล์เดียว

ส่วนที่ 3: ปุ่มในหน้าต่างใช้งานอย่างไร

Create Folder

- ใช้สร้างโฟลเดอร์ช่อง/เพจใหม่
- ไม่ต้องใส่ลิงก์ก็สร้างได้
- โปรแกรมจะเพิ่มหัวข้อ Channel Name ลงใน url_clip.txt ให้ด้วย

Add Links

- ใช้เพิ่ม URL เข้าไปในช่อง/โฟลเดอร์ที่เลือก
- เพิ่มแล้วจะไม่ลบข้อมูลช่องอื่น
- ถ้าวาง URL ซ้ำในช่องเดิม โปรแกรมจะไม่เพิ่มซ้ำ

Download All

- ใช้ดาวน์โหลดทุกช่องใน url_clip.txt พร้อมกัน
- คลิปของแต่ละช่องจะถูกบันทึกลงโฟลเดอร์ของช่องนั้น
- โปรแกรมจะแสดง log และสถานะระหว่างทำงาน

Clear Selected

- ใช้ล้าง URL / Status / Topic ของช่องที่เลือก
- โฟลเดอร์ยังอยู่
- คลิปวิดีโอที่ดาวน์โหลดไว้ไม่ถูกลบ

Delete Folder

- ใช้ลบช่อง/โฟลเดอร์ที่ไม่ใช้แล้ว
- ลบทั้งโฟลเดอร์จริงและข้อมูลช่องนั้นใน url_clip.txt
- คลิปในโฟลเดอร์นั้นจะถูกลบจริง
- ใช้เมื่อแน่ใจว่าไม่ต้องใช้ช่องนั้นแล้ว

Folder Status

ตารางสถานะจะแสดงข้อมูลแต่ละช่อง:

- Total = จำนวน URL ทั้งหมดในช่องนั้น
- Downloaded = จำนวนที่ดาวน์โหลดสำเร็จแล้ว
- Failed = จำนวนที่ดาวน์โหลดไม่สำเร็จ
- Pending = จำนวนที่ยังรอดาวน์โหลด
- Path = ตำแหน่งโฟลเดอร์จริง


ส่วนที่ 4: ดาวน์โหลดคลิป

วิธีหลักคือกดปุ่ม Download All ในหน้าต่าง Clip Batch Control

ถ้าต้องการรันด้วยคำสั่งเอง ยังใช้คำสั่งเดิมได้:

cd "C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels All\Facebook Reels to America"
py process_clip_batch.py

คำสั่งนี้จะทำงานให้โดยอัตโนมัติ:

- อ่าน url_clip.txt ทั้งไฟล์
- อ่านทุก Channel Name
- สร้างหรือใช้โฟลเดอร์ช่องเดิม
- ดาวน์โหลดคลิปของทุกช่อง
- ใส่ Status: Downloaded ให้คลิปที่โหลดสำเร็จ
- ใส่ Status: Failed ให้คลิปที่โหลดไม่สำเร็จ
- บันทึกคลิปลงโฟลเดอร์ที่ตรงกับ Channel Name

หลังดาวน์โหลดเสร็จ คลิปจะอยู่ในโฟลเดอร์นี้:

C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels All\Facebook Reels to America download clip\(Channel Name)


ส่วนที่ 5: วิเคราะห์คลิปด้วย Codex

หลังดาวน์โหลดเสร็จ ให้กลับมาที่ Codex แล้วพิมพ์:

analyze downloaded clip batch

Codex จะช่วยทำงานส่วนที่ต้องใช้ AI:

- ดูคลิปหรือสร้างภาพตัวอย่างชั่วคราวเมื่อจำเป็น
- เขียน Topic ภาษาไทย 1-2 บรรทัด
- ตั้งชื่อไฟล์ภาษาไทยแบบสั้นให้เข้าใจง่าย
- เปลี่ยนชื่อไฟล์เป็นรูปแบบ:

1-ชื่อสั้นของคลิป.mp4
2-ชื่อสั้นของคลิป.mp4
3-ชื่อสั้นของคลิป.mp4

ตัวอย่าง:

1-แปลงใต้บันไดเป็นมุมทำงาน.mp4
2-ทำแผงลายนูนพ่นสีเมทัลลิก.mp4
3-ทำแผ่นทางเดินจากฝาขวด.mp4

หมายเหตุ:

โดยปกติ โฟลเดอร์ช่องควรเหลือเฉพาะไฟล์ .mp4 เท่านั้น
ไม่จำเป็นต้องเก็บโฟลเดอร์ _frames หรือไฟล์ .info.json
ถ้า Codex ต้องใช้ไฟล์ช่วยวิเคราะห์ จะสร้างแบบชั่วคราวและลบทิ้งหลังใช้งาน


ส่วนที่ 6: ล้างหรือลบข้อมูลเก่าเมื่อใช้งานเสร็จ

แนะนำให้ล้างผ่านหน้าต่าง Clip Batch Control:

Clear Selected

- ล้าง URL / Status / Topic เฉพาะช่องที่เลือก
- เก็บโฟลเดอร์ไว้
- ไม่ลบคลิปวิดีโอ

Delete Folder

- ลบช่องที่เลือกออกจาก url_clip.txt
- ลบโฟลเดอร์จริงและคลิปในโฟลเดอร์นั้น
- ใช้เฉพาะช่องที่ไม่ต้องใช้อีกแล้ว

หมายเหตุ: การกด Delete Folder เป็นการลบไฟล์จริง ควรใช้เมื่อแน่ใจว่าใช้คลิปชุดนั้นเสร็จแล้ว


ส่วนที่ 7: วิธีใช้แบบสั้นที่สุด

ทุกครั้งที่ต้องการดาวน์โหลดชุดใหม่:

4. วาง URL
5. กด Add Links
6. ทำซ้ำกับช่องอื่นได้ตามต้องการ
7. กด Download All
8. หลังดาวน์โหลดเสร็จ กลับมาที่ Codex แล้วพิมพ์:

analyze downloaded clip batch

9. เมื่องานเสร็จ:

- ถ้าต้องการล้างเฉพาะข้อมูลช่องนั้น กด Clear Selected
- ถ้าต้องการลบช่องทิ้งทั้งหมด กด Delete Folder


ส่วนที่ 8: หมายเหตุสำคัญ

- ดาวน์โหลดเฉพาะคลิปสาธารณะ หรือคลิปที่คุณมีสิทธิ์ใช้เท่านั้น
- ไม่ควรใช้กับคลิปส่วนตัว กลุ่มปิด หรือคลิปที่ต้องข้ามระบบล็อกอิน/สิทธิ์เข้าถึง
- ถ้าดาวน์โหลดล้มเหลว อาจเป็นเพราะลิงก์ต้องล็อกอิน หรือแพลตฟอร์มจำกัดการเข้าถึง
- เครื่องที่ไม่มี Codex ยังสามารถรัน py process_clip_batch.py เพื่อดาวน์โหลดได้
- แต่การเขียน Topic ภาษาไทยและการตั้งชื่อไฟล์แบบเข้าใจเนื้อหา ยังควรใช้ Codex
