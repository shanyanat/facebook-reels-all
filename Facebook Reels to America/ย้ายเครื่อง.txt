 ขั้นตอนติดตั้งและรันโปรเจกต์บน Windows เครื่องใหม่

  ---
  ขั้นตอนที่ 1 — ดาวน์โหลดและติดตั้งโปรแกรมที่จำเป็น

  ต้องติดตั้ง 5 โปรแกรม ตามลำดับนี้

  ---
  โปรแกรมที่ 1: Python
  ดาวน์โหลดที่: https://www.python.org/downloads/
  - กดปุ่ม Download Python 3.x.x (เวอร์ชันล่าสุด)
  - เปิดไฟล์ที่ดาวน์โหลดมา
  - สำคัญมาก: ก่อนกด Install ต้องติ๊กถูกที่ช่อง "Add Python to PATH" ก่อน
  - กด Install Now รอจนเสร็จ

  ---
  โปรแกรมที่ 2: Warp Terminal
  ดาวน์โหลดที่: https://www.warp.dev
  - กด Download for Windows
  - ติดตั้งตามปกติ กด Next ไปเรื่อย ๆ

  ---
  โปรแกรมที่ 3: Google Chrome
  ถ้าเครื่องใหม่มี Chrome อยู่แล้ว ข้ามขั้นตอนนี้ได้เลย
  ดาวน์โหลดที่: https://www.google.com/chrome/

  ---
  โปรแกรมที่ 4: Claude Code
  ดาวน์โหลดที่: https://claude.ai/download
  - ติดตั้งตามปกติ กด Next ไปเรื่อย ๆ

  ---
  โปรแกรมที่ 5: Git
  (รายละเอียดการติดตั้งและตั้งค่าอยู่ใน ขั้นตอนที่ 4)

  ---
  ขั้นตอนที่ 2 — คัดลอกโฟลเดอร์โปรเจกต์มาจากเครื่องเก่า

  คัดลอกโฟลเดอร์ทั้งหมดนี้จากเครื่องเก่าไปวางในเครื่องใหม่:

  C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels to America\

  วิธีคัดลอก: ใช้ USB Drive, Google Drive, หรือ OneDrive ก็ได้ แล้ววางไว้ที่ตำแหน่งเดิม C:\Claude code\ปรึกษาส่วนตัว\ ในเครื่องใหม่

  ---
  ขั้นตอนที่ 3 — ติดตั้ง Python Packages (ทำครั้งเดียว)

  เปิด Warp Terminal แล้วพิมพ์คำสั่งเหล่านี้ทีละบรรทัด กด Enter ทุกครั้ง รอให้เสร็จก่อนพิมพ์บรรทัดถัดไป:

  คำสั่งที่ 1 — ติดตั้ง packages หลัก:
  py -m pip install playwright watchdog
  รอจนขึ้น Successfully installed... แล้วค่อยไปต่อ

  คำสั่งที่ 2 — ติดตั้ง Chrome สำหรับบอท:
  py -m playwright install chromium
  รอจนดาวน์โหลดเสร็จ (อาจใช้เวลา 2–3 นาที)

  ---
  ขั้นตอนที่ 4 — ติดตั้ง Git และตั้งค่า Warp

  ติดตั้ง Git (Windows เท่านั้น)
  ดาวน์โหลดที่: https://git-scm.com/install/windows
  - กด "Click here to download" แล้วติดตั้งตามปกติ

  ตั้งค่า Warp
  - เปิด Warp → เข้าเมนู Settings
  - เลือกแท็บ Features
  - หัวข้อ Default shell for new session → เลือก Bash.exe

  ---
  ขั้นตอนที่ 5 — ทดสอบรันโปรเจกต์

  เปิด Warp แล้วพิมพ์คำสั่งเหล่านี้ตามลำดับ:

  ไปที่โฟลเดอร์โปรเจกต์:
  cd "C:\Claude code\ปรึกษาส่วนตัว\Facebook Reels to America"

  ทดสอบดูว่าระบบทำงานได้:
  py bot.py status

  ถ้าเห็นตารางรายชื่อโปรเจกต์ขึ้นมา = ติดตั้งสำเร็จ พร้อมใช้งาน

  ---
  สิ่งที่ต้องทำเพิ่มครั้งแรกที่รันบอท

  ครั้งแรกที่รัน py bot.py images หรือ py bot.py videos บอทจะเปิด Chrome ขึ้นมา และจะให้ Login ChatGPT และ Google Flow
  ทำครั้งเดียว ระบบจะจำ Login ไว้ให้ครั้งต่อไปไม่ต้อง Login ใหม่
